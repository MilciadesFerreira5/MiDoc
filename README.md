# MiDoc

Para inicializar el proyecto:
```
npm i
npm run dev
```

# Librerias y Frameworks usadas:
```
npm install react-router-dom
npm i axios
npm install @mui/material @emotion/react @emotion/styled
npm install @fontsource/roboto 
npm install @mui/icons-material
```
